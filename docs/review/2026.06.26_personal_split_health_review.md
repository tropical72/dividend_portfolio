### [CRITICAL] 2.1. 개인별 소득세 과세(금융소득종합과세) 미분리 오류
- **코드 위치:** [projection_engine.py:L386-L394](file:///home/tropical72/Work/dividend_portfolio/src/core/projection_engine.py#L386-L394) 및 [projection_engine.py:L839-L843](file:///home/tropical72/Work/dividend_portfolio/src/core/projection_engine.py#L839-L843)
- **문제 현황:**
  대한민국 소득세법은 개인별 과세(부부합산 과세 안 함)가 기본 원칙입니다. 즉, 금융소득종합과세 기준(연 2,000만 원 초과 여부) 및 누진세율 적용은 부부 각각의 금융소득을 기준으로 따로 계산해야 합니다.
  그러나 현재 코드에서는 아래와 같이 **가구 전체의 합산 배당소득(`assessed_dividend`)과 가구 전체의 합산 외부 금융소득(`personal_external_financial_income`)**을 그대로 `calculate_us_dividend_tax` 함수에 전달하고 있습니다.
  ```python
  dividend_tax_detail = self.tax_engine.calculate_us_dividend_tax(
      assessed_dividend,
      other_financial_income=personal_external_financial_income,
      other_comprehensive_tax_base=personal_other_comprehensive_tax_base,
  )
  ```
- **영향 및 리스크:**
  `split_mode`가 `"couple"`인 2인 운용 모드이더라도, 종합과세 세액 계산 시 두 사람의 소득이 강제 합산되어 처리됩니다.
  예를 들어, 본인 배당 1,800만 원, 배우자 배당 1,800만 원(합계 3,600만 원)일 경우, 개별 기준으로는 둘 다 2,000만 원 이하이므로 종합과세 대상이 아니고 일반 원천징수세율(15.4%)로 종결되어야 합니다. 그러나 현재 엔진은 3,600만 원을 기준으로 금융소득종합과세를 부과하여 **세액을 엄청나게 과대평가(Overestimate)하는 치명적인 오류**를 범하고 있습니다.
- **해결 가이드라인 (Code Fix Guide):**
  2인 모드(`personal_split_mode == "couple"`)일 때는 각 owner(`self`, `spouse`)별로 배분 비율(`personal_owner_weights`)에 맞춰 배당금과 외부 소득을 나누고, `calculate_us_dividend_tax`를 각각 호출하여 그 세액 합계를 가구 총 세액으로 계산해야 합니다.
  *단, `other_comprehensive_tax_base`(기타 종합소득 과표)는 본인(self)에게만 적용하고 배우자는 `0.0`으로 인가하는 것이 합리적입니다.*
  ```python
  if personal_split_mode == "couple":
      tax_self = self.tax_engine.calculate_us_dividend_tax(
          assessed_dividend * personal_owner_weights["self"],
          other_financial_income=personal_external_financial_income * personal_owner_weights["self"],
          other_comprehensive_tax_base=personal_other_comprehensive_tax_base,
      )
      tax_spouse = self.tax_engine.calculate_us_dividend_tax(
          assessed_dividend * personal_owner_weights["spouse"],
          other_financial_income=personal_external_financial_income * personal_owner_weights["spouse"],
          other_comprehensive_tax_base=0.0,
      )
      # 두 세액 결과를 합산하여 dividend_tax_detail DTO 재구성
  ```

- **영향 및 리스크:**

  연산자를 strictly 초과(`>`)로 수정해야 합니다.
  ```python

      reflected_financial_income
      if reflected_financial_income
      > self.tax_engine.health_financial_income_threshold
      else 0.0
  )
  ```
